# Blockchain 类

该类提供了访问区块链数据的一系列方法。

命名空间：[FRS.SmartContract.Framework.Services.FRS](../FRS.md)

程序集：FRS.SmartContract.Framework

注：本文中标记 `new` 和 ` 已弃用 ` 的地方是 2.0 版本相对 1.6 版本的更改之处。

## 语法

```c#
public static class Blockchain
```
